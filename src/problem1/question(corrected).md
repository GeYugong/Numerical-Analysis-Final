# 数值分析期末大作业


一、飞行器遥测数据分析与控制系统 

1. 背景 

一架试验飞行器在返回阶段遭遇强电磁干扰，导致轨迹数据受损：下传的高度数据（Height）混入了传感器噪声，且在 $t = 6\text{s}$ 至 $t = 8\text{s}$ 期间发生丢包 。任务目标是根据现存数据建立运动模型，精准预测飞行器的理论落地时间，并解算姿态控制引擎的推力方程 。

2. 任务流程 

3. **数据恢复**：
    
    - 读取 `data.csv`，针对 $t=6, 7, 8$ 时刻的数据丢失情况，利用丢失点前后的有效数据。
		**针对 $t=6, 7$ 的插值**：选取支撑点 $t \in \{4, 5, 9\}$； **针对 $t=8$ 的插值**：选取支撑点 $t \in \{5, 9, 10\}$。
        
    - 选取合适的插值算法（拉格朗日或牛顿插值）填补这三个时刻的高度值 。
        
4. **模型构建与回归**：
    
    - 补全后的数据由于含有噪声，不能直接用于求导或外推 。
        
    - 在忽略空气阻力巨变的情况下，垂直运动近似满足二次函数关系 $h = at^2 + bt + c$ 。
        
    - 使用多项式回归算法分别进行线性拟合和二次拟合，并利用均方误差（MSE/RMSE）对比论证为何二次模型更适合，最终输出拟合得到的函数表达式 $h(t)$ 。
        
5. **落地时间预测**：
    
    - 求解飞行器接触地面的时刻（即 $h(t) = 0$） 。
        
    - 使用牛顿迭代法或割线法，以 $t=14$ 为初始猜测值进行求解 。
        
    - **思考题**：为什么此处不适合使用二分法？（提示：根可能在已知数据范围之外） 。
        
6. **引擎推力解算**：
    
    - 调整三个反推引擎 $(x_1, x_2, x_3)$ 以满足力学平衡线性方程组 ：
        
        $$\begin{cases} 4x_1 + 3x_2 = 24 \\ 3x_1 + 4x_2 - x_3 = 30 \\ -x_2 + 4x_3 = -24 \end{cases}$$
        
                
    - **要求**：使用**超松弛迭代法 (SOR)** 求解，取 $\omega = 1.25$，$x_0 = (1, 1, 1)^T$，设置容差 $tol = 1e-6$，并输出收敛所需的迭代次数 。
